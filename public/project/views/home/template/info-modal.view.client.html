INFORMATION: Organizer

  • The application has two user roles: Being an Organizer is one of them.
  • If you are a first time user and want to organize an event, register to the application as an Organizer.

    If you are revisiting, you can login. more info
  • As an Organizer you'll be able to see all the events that are created by you.
    If your event is happening today, it's shown with a green marker on the map, red if it's in the future and blue if it's past dead.
  • Add a new event.more
    Enter the event name, give a brief description for your event, enter the date of event, specify the location where this event will be held.
  • Add checkpoints for event.more
    For an event, place multiple checkpoints each leading to the next one, enter the checkpoint info/riddle and choose a location.
  • See registered teams.more
    You can see who all registered with your event.
    On the map you can see the teams progress and the last know location of each team member

INFORMATION: Participant

  • The application has two user roles: Being a Participant is one of them.
  • Home page shows all the events around you.
    You can also search for specific place and find events.

    If you are a first time user and want to participate in an event, register to the application as a participant.

    If you are revisiting, you can login. more info
  • As a Participant you can switch between viewing all the events and the events you have registered to.
    If your event is happening today, it's shown with a green marker on the map, red if it's in the future.
  • Registering to an event.more
    Click on the unregistered marker event you want to participate in, click on register and give a team name.
    If you want to join a team created by other participant/s you can search by team or you can search for a specific participant by username and join their team.
  • Start an event.more
    • Click on the registered marker event you want to start (one of the green marker/s).
    • If this is the first time you have started the event, you'll be able to see the first checkpoint.
    • The information/riddle in the previous checkpoint will lead you to the next checkpoint.
    • Click on "Am I here" button, if you think you have arrived at the next checkpoint.
    • The application reveals the checkpoints once you are within 100m of the checkpoint.
    • TThe last checkpoint will have the information that you have finished the event.
    • Unfortunately there is no winner team with this application currently, but the idea is to get on the road, team up, explore and have fun.
    • Happy Treasure-hunting...
